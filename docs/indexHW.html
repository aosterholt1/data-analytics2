    • Summary

    The purpose of this project was to analyze how weather changes depending on how far away from the equator you are. To accmplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

    After assembling the data, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included temperature, cloudiness, humidity, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
